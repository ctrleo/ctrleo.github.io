Privacy Policy

Pertaining to all sites under ctrleo.github.io and it's affiliated backends/APIs

Data Usage

No user data is kept, stored or shared by ctrleo, or any other owners of this site besides the entities mentioned below:

Spotify (Spotfy API) in (ctrleo.github.io/taylorizer)

We (the site) use Spotify's web API to authenticate user's and read and write private and public playlists

Spotify API Scope

The app (ctrleo.github.io/taylorizer) accesses the following Spotify data (as permitted by Spotify API Scope)

  • Read User's Private Playlists (permitted by scope 'playlist-read-private')
  • Modify User's Private Playlists (permitted by scope 'playlist-modify-private')
  • Modify User's Public Playlists (permitted by scope 'playlist-modify-public')

The app cannot and will not access any type of user data outside of the scope and doesn't collect, share or store any user data it accesses, such as private playlists and contents

Using Spotify's App pertains to their own privacy policy

Read Spotify's Privacy Policy (UK) and Spotify's Privacy Policy (US)
